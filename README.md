A console-based Personal Finance Tracker application developed in Java that allows users to manage their financial transactions, including income and expenses, using full CRUD (Create, Read, Update, Delete) operations.

This project focuses on applying core Java fundamentals, particularly the use of Lists (ArrayList) for state management, Encapsulation for data integrity, and Input Validation to ensure a robust and interactive user experience. The application runs in a continuous menu-driven loop, enabling users to add, view, update, delete transactions, and calculate their overall financial balance.
